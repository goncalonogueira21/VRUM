# VRUM


To run VRUM in your localhost you have to execute:

If first time:

	install python
	
	install pip
	
	pip install -r requirements.txt
	
	Start Mongo
	
	export FLASK_APP=vrum.py
	
	flask run

else:

	Start Mongo
	
	export FLASK_APP=leonardo.py
	
	flask run

Windows:

	por o mongo a correr
	
	set FLASK_APP=vrum.py

	flask run
